Jamba Release Notes / News


Whats New:


Cross Realm Support Added

* Jamba 4.0+ cross realm & connected realms character support added.

* Character names now must be in name-server format, .e.g. Ebony-Outland


Notes:

CR = Connected Realms - EG You can trade/join guild ETC. Jamba will function as though your team is on the same realm!

CR = CrossRealm - Join a realm over CRZ tech. Note: Some modules in Jamba will not function or will not work correctly while in CRZ. This is due to limitations in Blizzard's API. You can talk over realms with addons, however YOU NEED TO BE IN PARTY WITH THE MEMBER FOR ANYTHING TO WORK!


Automated Offline System Added:


* Jamba will now automatically set minions offline if the minion is not online. Jamba will dectect when the minion comes online and sets the status to online. You can now have a full list of characters and not have to worry about online/offline. Easy switching!


Jamba-QuestWatcher:


* Supports Bonus Objectives.


Having Problems With Your Teams?


* Remember to do the following fixes on every character in your team!

* If your team list is full of empty lines, please reset the current profile. Warning this will require you to configure everything again. You can reset the current profile with the following button: /jamba > Profiles > Core: Profiles > Reset Profile

* Make sure to add characters into your team list with their name, followed by a dash and the server the character is on. e.g. Ebony-servername


ISBoxer Users


* Please make sure your ISBoxer version is ISBoxer 41.10.1115.1 or higher.

* When ISBoxer manages your Jamba Team for you, it will add the characters with the correct name format if you have filled out the server name in your ISBoxer character.

*Jamba will remove any spaces Isboxers trys to add. No realm can have a space in the name.
